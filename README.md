# Deepali Creation - POS & Inventory Management System

A premium, full-stack POS and Inventory Management web application custom-built for Deepali Creation.

## Tech Stack
- **Frontend**: React, Redux Toolkit, TailwindCSS/Vanilla CSS, Lucide icons, Vite.
- **Backend**: Node.js, Express, MongoDB, Mongoose, JWT authentication.
- **Additional Utilities**: Barcode generation via `bwip-js`, multi-layout printable sticker templates, and Excel reports exports.

## Key Features
1. **Sales & Billing (POS)**: Real-time checkout, barcode scanning support, tax & discount processing, receipt printout.
2. **Product Catalog**: Barcode assignment, stock alerts, category/brand classifications, and dynamic selling price updates.
3. **Bulk Barcode Generation**: Multi-layout sheet selector (up to 40 labels per sheet) with real-time style updates.
4. **Excel Exports**: Row selection and bulk exports for products catalogue and sales ledger history.
5. **Inventory Valuation**: Current stock quantity tracking, Capital Investment tracking, and retail value projection reports.
6. **Role-Based Access Control**: Cashier and Warehouse roles restricted from editing/deleting critical databases.
